# Portafolio App

Portfolio personal desarrollado con [Astro](https://astro.build) y [Tailwind CSS v4](https://tailwindcss.com). Diseño oscuro con estética cyberpunk/neon, completamente responsivo y bilingüe (es/en).

## Stack

- **Framework:** Astro 5
- **Estilos:** Tailwind CSS v4
- **Fuentes:** Inter + Space Grotesk
- **Iconos:** Material Symbols
- **Despliegue:** —

## Proyectos incluidos

- SistBienes — Gestión de Activos
- BeFit — AI Wellness Companion
- SGS — Sistema de Gestión Social
- DocFlow — Visual Document Design Engine

## Comandos

| Comando | Acción |
|---------|--------|
| `npm install` | Instalar dependencias |
| `npm run dev` | Iniciar servidor local |
| `npm run build` | Build producción → `./dist/` |
| `npm run preview` | Preview del build local |
